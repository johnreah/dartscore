### Python notes
[Python tutorial on virtual environments and Packages](https://docs.python.org/3/tutorial/venv.html)

Useful python commands (for Windows):
* python -m venv .venv # creates virtual environment in .venv directory
* .venv\Scripts\activate # activates virtual environment
* deactivate # deactivates virtual environment
* python -m pip install PyQt6 # installs PyQt6 module
* python -m pip freeze > requirements.txt
* python -m pip install -r requirements.txt
